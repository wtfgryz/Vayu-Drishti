**Vayu-Drishti** (Sanskrit for *Air Vision*) is an end-to-end Big Data Analytics pipeline designed to monitor, analyze, and forecast Air Quality Index (AQI) levels for major Indian metropolises.

## ğŸš€ Project Overview
This project demonstrates the **4 V's of Big Data** (Volume, Velocity, Variety, Veracity) using real-world sensor data from the Central Pollution Control Board (CPCB).

* **Ingestion:** Automated hydration of Data Lakes using CPCB datasets.
* **Processing:** ETL pipelines using **Apache Spark** to handle massive sensor logs.
* **Analytics:** Machine Learning (**Linear Regression**) to forecast AQI trends through 2026.
* **Visualization:** Interactive command-center dashboards using **Plotly**.
